My Favorite Hobby Coding

My favorite hobby is coding.Coding is a skill where you take instructions (the steps in a task) and translate it into a language the computer understands since computers do not communicate like humans. They communicate in a language called BINARY and it is uses 0's and 1's. Coders write the instructions using a programming language.Coding means using the programming language to get the computer to behave as desired. Each line of the code is a set of instructions for the computer. A set of codes form a script, and a set or dozens of sets, form a program.
